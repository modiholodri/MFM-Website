Movie File Merger Lists Tab

The Lists tab is the main tab to combine video collections. Use it to move items around in the different lists and sort your collection.

Movie File Merger Maintenance Tab

The Maintenance tab is used to cleanup your newly acquired videos and add-on files or to cleanup your collection in general. Use it in conjunction with the Windows Explorer or FileBot, but basically, you can use it with any program, which accepts file drops.

Movie File Merger Log

The Log tab shows basic information about what has been done during this session. After a Just Scan It run has been performed it shows also duplicated videos.

Movie File Merger Settings

The Settings tab contains all settings to adjust Movie File Merger to the style of your collection. Anyway, if you follow my style, then there should not be too much need to change any of the settings.

Movie File Merger Instructions

The Instructions tab contains brief instructions about how to use MFM. For a more detailed description just click on any of the book logos shown in the MFM program.

Movie File Merger About

The About tab contains the version information and copyright. You can change the frequency with which MFM checks for new versions.
